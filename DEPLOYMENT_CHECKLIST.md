# 🚀 Grappl 배포 전 최종 체크리스트

배포하기 전에 다음 항목들을 순서대로 확인하여 서비스의 안정성을 확보하세요.

## 1. ✅ 최근 수정 사항 검증 (Critical Fixes)
가장 최근에 해결한 버그들이 확실하게 고쳐졌는지 확인합니다.

- [ ] **비로그인 드릴 목록 로딩**: 시크릿 모드(로그아웃 상태)에서 `/drills` 페이지 접속 시 무한 로딩 없이 목록이 잘 뜨는지 확인.
- [ ] **드릴 클릭 로그아웃 방지**: 로그인 상태에서 드릴 카드를 클릭했을 때 로그아웃되지 않고 상세 페이지로 이동하는지 확인.
- [ ] **오늘의 미션 초기화**: 자정(Local Time 00:00)이 지났을 때 퀘스트가 정상적으로 초기화되는지 확인 (코드 레벨에서 수정됨).
- [ ] **보너스 드릴 표시**: 보너스 드릴이 포함된 코스 상세 페이지에서 '보너스 드릴' 탭이 보이고 목록이 로딩되는지 확인.
- [ ] **인스트럭터 삭제**: 관리자 페이지에서 인스트럭터 삭제 시, 관련 코스/비디오/드릴이 함께 삭제되며 에러가 발생하지 않는지 확인.

## 2. 🔐 인증 및 사용자 관리 (Auth & User)
- [ ] **Google 로그인**: 구글 로그인 버튼 클릭 시 정상적으로 로그인되고 세션이 유지되는지 확인.
- [ ] **이메일 가입/로그인**: 신규 이메일 가입 및 로그인이 정상 작동하는지 확인.
- [ ] **프로필 수정**: 마이페이지에서 닉네임, 벨트 레벨 수정이 반영되는지 확인.

## 3. 💳 결제 및 구독 (Payments)
- [ ] **Stripe 연동**: 결제 시도 시 Stripe 테스트 결제창이 뜨는지 확인.
- [ ] **구독 권한**: 구독 후 '구독자 전용' 콘텐츠에 접근 가능한지 확인.
- [ ] **단건 구매**: 코스 단건 구매 후 '내 라이브러리'에 즉시 추가되는지 확인.

## 4. 🛡️ 데이터베이스 및 보안 (DB & Security)
- [ ] **RLS 정책 적용**: `force_public_access.sql` 스크립트가 Supabase에서 실행되었는지 확인 (비로그인 접근 권한 필수).
- [ ] **환경 변수**: `.env` 파일의 `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`가 배포 환경(Vercel 등)에도 설정되었는지 확인.

## 5. 📱 UI/UX 및 반응형
- [ ] **모바일 뷰**: 모바일 화면에서 네비게이션 바, 드릴 목록, 영상 플레이어가 깨지지 않는지 확인.
- [ ] **로딩/에러 처리**: 데이터 로딩 중 스피너가 표시되고, 에러 발생 시 적절한 메시지가 나오는지 확인.

## 6. 📦 빌드 및 배포 (Build & Deploy)
- [ ] **로컬 빌드 테스트**: 터미널에서 `npm run build` 명령어가 에러 없이 완료되는지 확인.
- [ ] **Vercel 설정**: `vercel.json` 설정이 올바른지, 배포 후 라우팅(새로고침 시 404 에러 방지)이 잘 되는지 확인.

---
**💡 팁**: 문제가 발생하면 브라우저 개발자 도구(F12)의 `Console` 탭을 먼저 확인하여 에러 메시지를 캡처해두세요.
